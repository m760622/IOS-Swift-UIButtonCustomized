# IOS-Swift-UIButtonCustomized
``` Level : Beginner```

This tutorial Learns the easy way to Add Border , Shadow , Image Icon to Button and change Color in Xcode by Swift Program ( Custom UIButtons & UIColors). This tutorial is made with Xcode 10 and built for iOS 12.

## About

Sample app to How To Create Rounded Button in Xcode ( Custom Buttons). 

## Requirements

- iOS 12.0+
- Xcode 10.0+
- Swift 4.0+
- iPhone X+

## Usage

You won't need to Download any extra libaray just ```import UIKit``` inside the project
